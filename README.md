## Dependencias

-  g++
-  ffmpeg
-  ncurses
-  SDL
-  sdl2_image

## Comandos

-  Limpiar: `make clean`
-  Compilar: `make all`
-  Ejecutar: `make run`
-  Limpiar + compilar + ejecutar: `make`
